# Manifest of external skill repos to clone + pull on update.
#
# Format: url|branch|path
#   url:    git remote (https or ssh)
#   branch: branch to check out at initial clone time (install.sh only)
#   path:   directory name under skills-repos/
#
# install.sh clones missing repos at the configured branch, skips existing.
# update.sh pulls the CURRENTLY checked-out branch with --ff-only,
# so it's safe even if you have uncommitted local edits — it just skips
# with a warning instead of clobbering work.
#
# Lines starting with # and blank lines are ignored.
