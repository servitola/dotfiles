# Your site

This folder contains your site. You do not edit anything here by hand —
just chat with the AI, and it puts your content in the right place.

## What you can do

- **Send a photo** → say "this is for the gallery", "this is for the
  post", "this is for my profile" — the AI files it.
- **Write a blog post** → send text (by voice or written) and say "new
  post". The AI formats and saves it.
- **Show your work** → send a description plus images and say "this is
  for portfolio".
- **Change the look** → "make the background darker", "change the main
  color to green", "rename my site".
- **Publish** → say "publish" or "update the site". A link comes back
  in a minute or two.

This README is for the AI's reference. Ask the AI anything in your own
language — it translates and acts.
